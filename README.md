# @blockether/foundation-react

React-based chat UI foundation with integrated SQL analysis capabilities.

## Features

- **Chat Interface Foundation**: Extensible, unstyled React components for building chat interfaces
- **SQL Integration**: Full-featured SQL editor with DuckDB-WASM for in-browser data analysis
- **Custom Section Renderers**: Flexible ChatArea with customizable sections (conversation, headers, sidebars)
- **AG-UI Protocol Compatible**: Seamless integration with AI agents using standardized event-based architecture
- **File Resolver Support**: Custom resolvers with metadata for intelligent column descriptions
- **Performance Optimized**: Renders 1000+ messages at 60fps with virtual scrolling
- **Fully Accessible**: WCAG compliant with comprehensive ARIA support
- **Type Safe**: Complete TypeScript coverage with strict mode

## Installation

```bash
npm install @blockether/foundation-react
# or
pnpm add @blockether/foundation-react
# or
yarn add @blockether/foundation-react
```
